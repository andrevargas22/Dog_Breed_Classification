Dog Breed Classifier with Machine Learning


This web app runs my final project for Udacity's Machine Learning Engineer Nanodegree.

Upload a dog image and the Machine Learning Model will try to classify it's breed. If it's a human image, it will say which dog breed the person resembles the most. The model has an 88% accuracy.
